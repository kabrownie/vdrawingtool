# Contributing to DrawingTool Website

Thank you for considering contributing to the DrawingTool website! We appreciate your help in making this project better.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix: `git checkout -b feature/your-feature`.
3. Make your changes and commit them: `git commit -m 'Add new feature'`.
4. Push to your branch: `git push origin feature/your-feature`.
5. Open a pull request to the `main` branch.

## Reporting Bugs

If you find a bug, please open an issue and provide as much detail as possible, including steps to reproduce.

## Feature Requests

Feel free to open an issue to suggest new features or improvements.

## Code Style

Follow the existing code style and conventions in the project. Consider running linting tools before submitting your changes.


Happy coding!
